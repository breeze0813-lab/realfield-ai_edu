STEP 1

본인 학원 정보 5가지 입력 · 3분

완벽하지 않아도 OK. 1~2줄씩만. 입력하면 아래 시스템 프롬프트가 자동 완성됩니다.

예: 티포인트에듀, 초·중·고 수학·영어, 강동구 고덕동
따뜻함·전문성·엄격함·친근함 등에서 선택하거나 본인 학원 톤을 묘사
학원 슬로건·자주 쓰는 한 줄·강조 단어
학원 톤과 안 맞는 단어·과장 표현 등
"우리 학원은 ___ 학원이다"의 빈 칸을 채워주세요
STEP 2

Claude / ChatGPT에 저장 · 8분

아래 자동 생성된 시스템 프롬프트를 복사해서 Claude Project 또는 ChatGPT Custom GPT의 *Instructions* 칸에 붙여넣고 저장하세요.

위 5가지를 입력하시면 여기에 시스템 프롬프트가 자동 완성됩니다.
Claude Project 만드는 법
ChatGPT Custom GPT 만드는 법
  1. claude.ai 접속 → 로그인
  2. 왼쪽 사이드바 Projects 메뉴 클릭
  3. 오른쪽 위 + New project 버튼 클릭
  4. 프로젝트 이름 입력 (예: T-Point Edu 비서)
  5. Add custom instructions(또는 Set custom instructions) → 위에서 복사한 프롬프트 붙여넣기
  6. Save 또는 Create 클릭 → 완료
  1. chatgpt.com 접속 → 로그인 (유료 플랜 필요)
  2. 왼쪽 사이드바 GPTs 탐색 또는 Explore GPTs 클릭
  3. 오른쪽 위 + Create 클릭
  4. Configure 탭으로 이동 → 이름·설명 입력
  5. Instructions 칸에 위에서 복사한 프롬프트 붙여넣기
  6. 오른쪽 위 Create → 본인만 사용으로 저장
STEP 3

세팅 전 vs 후 비교 체험 · 3분

아래 질문 1가지를 골라 — 일반 Claude/ChatGPT와 방금 만든 Project/GPT에 각각 던져보세요. 차이를 직접 느껴보세요.

    Claude 무료 계정인데 Project 못 만들어요

    Claude.ai 무료 계정도 Project 가능합니다. 왼쪽 사이드바 [Projects] 메뉴 확인. 안 보이시면 *Claude.ai에 로그인된 상태인지* 먼저 확인하세요. ChatGPT 유료 계정 있으시면 Custom GPT로 만드셔도 동일한 기능입니다.

    회사·학원 컴퓨터에서 Claude/ChatGPT 차단되어 있어요

    휴대폰으로 하시거나, 옆 자리 분과 같이 보세요. 집에 가서 개인 컴퓨터로 다시 만드시면 됩니다. 입력하신 5가지 내용은 *이 페이지 자동 저장*되어 있으니, 같은 브라우저로 다시 열면 그대로 있습니다.

    5가지 정보 너무 짧게 적었는데 괜찮나요?

    1줄씩만 적어도 OK. 시작은 짧게, 쓰면서 점점 추가하시면 됩니다. 완벽한 1번보다 시작한 1번이 100배 좋습니다.

    이거 매번 새로 만들어야 해요?

    아니요. Project 1개에 모든 학원 정보를 누적하시는 게 핵심입니다. 분야별로 여러 개 만들기보다, 1개 큰 Project에 점점 추가하세요.

    강사들도 이거 만들어줘야 해요?

    본인 것부터 만들고 잘 작동하는 게 확인되면, 강사용 공용 Project를 따로 만드시는 게 좋습니다. 강사용은 학원 정보 + 응대 톤까지만, 본인 것은 모든 영역 포함.

    입력한 내용은 어디에 저장되나요?

    이 페이지의 로컬 저장소(이 브라우저 안)에만 저장됩니다. 서버로 전송 안 됨. 다른 컴퓨터·브라우저에서는 안 보입니다. 시크릿 모드·캐시 삭제 시 사라집니다.
